Overview

This page demonstrates how to build a simple, semantically correct HTML5 document that supports accessibility and SEO best practices.

Semantic Structure

Using semantic tags like <header>, <nav>, <main>, <section>, and <footer> helps both users and search engines understand your content.

Accessibility

Landmark elements and attributes like aria-label improve navigation for screen readers. Use clear headings and descriptive link text.

SEO Tips

  • Use a descriptive <title> and <meta> tags.
  • Structure headings hierarchically: <h1> to <h6>.
  • Include relevant keywords naturally in your content.
